## Stackroot 0.2.5

Download and run `Stackroot-Setup-0.2.5.exe` on **Windows 10/11 (64-bit)**. Update over **0.2.4** — sites, databases, and settings are kept.

### Reliability

- **Steadier RAM over long sessions** — background port lookups and header metrics no longer let stale cache entries accumulate; memory stays closer to a flat line during all-day use.
- **Lighter port checks** — fewer redundant process lookups when a recent port result is still valid.
- **Performance page cleanup** — leaving Performance releases its last full snapshot instead of holding it in the background.

### Dashboard

- **Uptime tooltips** — fixed a handler leak that could keep tooltip timers alive after rows were recycled.

### Install

Download `Stackroot-Setup-0.2.5.exe` below and run the installer. **Close any running Stackroot instance** (tray → Quit) before upgrading.

### Upgrading from 0.2.4

Your sites, databases, and settings are preserved. No extra steps — quit, install, and reopen Stackroot once.
